## ▌ PROJECT OVERVIEW

An enterprise-grade, zero-dependency real-time system telemetry engine engineered for ultra-low overhead performance auditing on Linux and Unix environments. It consolidates critical kernel metrics directly inside an active terminal session, bypassing bloated tracking daemons.

> **Design Goal:** Eradicate diagnostic overload via clean, asymmetric Left-Accent visual resource load cards (`▌`) and microsecond execution bounds.

*   **Zero Dependencies:** Runs natively via core POSIX utilities (`awk`, `sed`, `grep`, `ps`, `df`, `vmstat`).
*   **Observability First:** Instantaneous interpolation of raw kernel vectors into responsive dashboard modules.
*   **Dual-Operational Profile:** Supports user-guided interactive loops or automated script pipeline execution.
